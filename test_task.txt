Тестовое задание: Построить простейшие приложение на nodejs.

Требования
1. Приложение должно иметь простейший сервер с использованием framework express (nodejs не ниже версии 8).
2. Систему авторизации пользователя (можно без использования БД для хранения данных авторизации).
3. Страница с списком всех данных.
4. Страницу с формой создания записи данных.
5. Страница редактирования данных.
6. Формат хранимых данных: идентификатор, название, текст, дата.
6. Данные должны сохраняться в БД (СУБД PostgreSQL или mongodb).
7. Для страниц использовать шаблонизатор pug или отдельное приложение одностраничник на vue 2.
